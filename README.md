# Сайт автосалона Premium Cars

Многостраничный сайт для автосалона с каталогом автомобилей, фильтрацией и страницами отдельных предложений.

## Технологии

- HTML
- CSS/SCSS
- JavaScript (Vanilla)
- Cloudflare Pages (для деплоя)

## Структура проекта

```
├── index.html          # Главная страница
├── catalog.html        # Каталог автомобилей
├── car.html           # Страница автомобиля
├── services.html      # Услуги салона
├── credit.html        # Кредит и лизинг
├── trade-in.html      # Trade-in
├── about.html         # О компании
├── faq.html           # FAQ
├── contacts.html      # Контакты
├── policy.html        # Политика конфиденциальности
├── css/
│   └── main.scss      # Основные стили
├── js/
│   ├── data.js        # Данные автомобилей
│   ├── components.js  # Компоненты (header, footer)
│   ├── main.js        # Главный скрипт
│   ├── catalog.js     # Логика каталога и фильтров
│   ├── car-detail.js  # Страница автомобиля
│   ├── credit.js      # Калькулятор кредита
│   └── faq.js         # FAQ аккордеон
└── package.json       # Зависимости для сборки SCSS
```

## Функциональность

- ✅ Каталог автомобилей с фильтрацией (марка, цена, год, тип кузова)
- ✅ Страница отдельного автомобиля
- ✅ Логика "подобных предложений"
- ✅ Формы заявки и обратной связи
- ✅ Калькулятор кредита
- ✅ Адаптивный дизайн
- ✅ Мобильное меню

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Скомпилируйте SCSS в CSS:
```bash
npm run build-css
```

Или для автоматической компиляции при изменениях:
```bash
npm run watch-css
```

3. Откройте `index.html` в браузере или используйте локальный сервер.

## Деплой на Cloudflare Pages

1. Подключите репозиторий к Cloudflare Pages
2. Укажите команду сборки: `npm run build-css`
3. Укажите директорию вывода: `/` (корень проекта)
4. Cloudflare Pages автоматически задеплоит сайт

## Особенности

- Все данные хранятся в `js/data.js`
- Компоненты header и footer генерируются динамически
- Фильтрация работает на клиентской стороне
- Формы готовы к интеграции с backend API
